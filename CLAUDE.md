# Project Documentation

## Agent skills

### Issue tracker

Issues are tracked as markdown files in `.scratch/` directory. See `docs/agents/issue-tracker.md`.

### Triage labels

Issues use standard triage labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout with `CONTEXT.md` and `docs/adr/` at repo root. See `docs/agents/domain.md`.
